# エラーコード -102 の対処法

## エラーコード -102 とは

エラーコード -102 は、通常ネットワーク接続エラーを示します。Firebaseへの接続に問題がある可能性があります。

## 確認すべきポイント

### 1. インターネット接続
- インターネットに接続されているか確認
- ファイアウォールがFirebaseへの接続をブロックしていないか確認

### 2. Firebase設定
- Firebase設定情報が正しく入力されているか確認
- `frontend/composables/useAuth.ts` の `firebaseConfig` を確認

### 3. ブラウザのコンソールでエラー詳細を確認
1. ブラウザで `http://localhost:3000/register` にアクセス
2. F12キーを押して開発者ツールを開く
3. 「Console」タブを開く
4. エラーが発生した際に、詳細なエラーメッセージを確認

### 4. 開発サーバーの状態
- フロントエンドサーバー（Nuxt.js）が起動しているか確認
- バックエンドサーバー（Laravel）が起動しているか確認

## 解決方法

### 方法1: ブラウザをリロード
ページをリロード（F5 または Cmd+R）して、再度試してください。

### 方法2: 開発サーバーを再起動
```bash
# フロントエンドサーバーを停止（Ctrl + C）してから
cd frontend
npm run dev
```

### 方法3: Firebase設定を確認
`frontend/composables/useAuth.ts` の `firebaseConfig` が正しく設定されているか確認してください。

### 方法4: ブラウザのキャッシュをクリア
ブラウザのキャッシュをクリアしてから、再度アクセスしてください。

## エラーメッセージの表示

エラーコード -102 が発生した場合、以下の日本語メッセージが表示されます：

**「ネットワーク接続エラーが発生しました。インターネット接続とFirebase設定を確認してください」**

このメッセージが表示された場合は、上記のポイントを確認してください。
